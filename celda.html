Robot Seleccionado

ABB IRB6700

La aplicación deseada requiere de dos factores importantes, un buen alcance horizontal y una capacidad de carga relativamente alta, ya que se requiere manipular ruedas de automóvil, las cuales pesan al rededor de 25-30 kg para automóviles convencionales hasta 50kg para camionetas y camiones.

Por otro lado, debido a al configuración de la celda, se requiere un amplio alcance horizontal, al rededor de 2m. Se llegó a este valor luego de realizar un montaje inicial de modelos CAD.

Por las razones anteriores, se optó por un manipulador ABB IRB 6700, el cual tiene un alcance horizontal de 2.6m y una capacidad de carga de 200 kg.

IRB 6700

CADs usados

Se usaron principalmente dos modelos CADs. Uno de el vehiculo al cual se le ensamblarán las ruedas y otro modelo CAD para las mismas.

Lógica Estación

La lógica de la estación consiste de dos componentes inteligentes y un controlador. Como entrada de este ultimo, se tiene una señal proveniente desde la celda donde se generan las ruedas de los carros. Como salidas del controlador, se tienen las señales para activar y desactivar la ventosa en el robot, reactivar el conveyor y activar la ventosa en el carro. El componente Inteligente Celda Rueda tiene como entradas las señales para activar y desactivar la ventosa del robot y una señal de verificación que indica si hay presente un carro listo para ponerle las ruedas. El componente inteligente Celda Carro tiene como entradas las señales para activar la ventosa en el carro y la de reactivar el conveyor provenientes desde el controlador, y como salida tiene una señal que le indica al componente Celda Rueda si hay un carro presente.

Lógica Celda Rueda

Este elemento se encarga de controlar la alimentación de las ruedas al robot y de activar o desactivar su ventosa.

Lógica Celda Carro

Este elemento se encarga de controlar la llegada de nuevos carros luego de que se les ha puesto las ruedas.
